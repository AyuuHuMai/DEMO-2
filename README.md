# My Dream Gaming Setup 💻🎮

**Second Project** | Pure HTML Website

A simple and clean static website that showcases different Gaming PC setups according to budget.

## 🌟 Project Overview

This is my second web development project. The website provides complete information about **Budget**, **Mid-Range**, and **High-End** gaming setups with images, descriptions, price ranges, and direct Amazon purchase links.

## ✨ Features

- Built using only HTML5 (No external CSS used)
- Three Gaming Setup Categories:
  - Budget Setup (GTX 1650)
  - Mid-Range Setup (RTX 3060)
  - High-End Setup (RTX 4090)
- Working Amazon product links
- Comparison table
- Images with captions and links
- Clean and easy-to-read layout

## 🛠 Technologies Used

- HTML5
- Semantic HTML elements (`<article>`, `<figure>`, `<table>`, etc.)
- Minimal inline styling

## 📁 Project Structure
DEMO-2/ ---- index.html ---- GTX 1650.jpg ---- RTX 3060.jpg ---- RTX 4090.jpg ---- README.md

## 🚀 How to Run

1. Download or clone this repository
2. Open the `index.html` file in any web browser
3. All images and links will work properly

## 📌 About This Project

This project helps gamers understand different PC setups according to their budget. It includes detailed information about popular NVIDIA GPUs with real purchase links.
